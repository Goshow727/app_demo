## AI视频生成生成管理系统前端页面；

## 运行环境
  运行需要使用docker进行部署。
  docker-compose配置文件参考 分支 docker
## 软件基础架构
  vue 3.x
  typescript
  element-plus
  pinia
  vue-router 4.x
## 启动项目
  安装依赖 npm i
  npm run dev 启动项目
  访问项目 http://localhost:6688
  系统默认：登录账号：admin 密码：123456
## 打包项目
  npm run build 打包项目
  打包完成后，项目会生成 dist 目录，目录下包含了项目的所有静态资源和打包后的代码。
